Large Action Models — なぜ今、ロボットが急に賢くなったのか
📖 約5分 🌱 Phase 1 / Issue 001

masa、今日ヤバいネタ見つけた。ロボットが食洗機を自分で回し始めた。しかも人間ノータッチで。

🔎 何が起きてるのか

ChatGPT以来、AIって「文章を書くもの」だったよね。でも2026年、その前提が崩れてきてる。AIがキーボードの向こう側から飛び出して、物理世界で実際に動き始めた。

鍵になってるのがVLA（Vision-Language-Action）モデル。「見て・理解して・動く」を1つのAIでやる技術。業界では「Large Action Models」とも呼ばれてて、LLMの次のパラダイムとして一気に注目が集まってる。

LLM = テキスト入力 → テキスト出力
VLA = 映像＋テキスト入力 → 物理的な動作を出力

AIの出力が「言葉」から「行動」に変わった。これが革命的な違い。
🤖 Figure 02の「61手」

一番わかりやすい事例がFigure AI社のデモ。ヒューマノイドロボットが食洗機から食器を出して、キッチン内の所定の場所に並べて、また戻して食洗機を起動する — これを完全自律でやった。

61
連続自律動作の数。歩行・認識・把持・配置・ドア操作まで、人間の介入ゼロ。ヒューマノイド史上最長・最複雑な自律タスク。

驚くのは、手だけじゃなく身体全体を道具として使ったこと。引き出しを腰で閉めて、食洗機のドアを足で持ち上げた。人間が無意識にやってる動きを、AIが勝手に学んでる。

しかもFigure社は24時間365日稼働を達成済み。バッテリーが減ると自分でドックに行って、充電済みの別ロボットと交代する。「疲れない工場」がもう現実になってる。

🔴 NVIDIAの仕掛け

この流れを裏で加速させてるのがNVIDIA。Isaac GR00T N1.6っていうヒューマノイド専用の基盤モデルを、しかもオープンソースで出した。TechCrunchは「ロボティクスのAndroid」と呼んでる。

NVIDIAはGPUで「AI学習のインフラ」を支配した。次は「AI行動のインフラ」を押さえようとしてる。モデル＋チップ＋シミュレーション環境をセットで提供して、どのロボットメーカーもNVIDIAの上に乗る構図を作ろうとしてる。GPUの再現だね。

📊 学会も一色

学術界も同じ。トップ学会ICLR 2026ではVLA関連の論文が164本出た。2024年にはほぼゼロだった分野が、2年でメインストリームに。

中国のAnt Group（アリペイの親会社）はLingBot-VLAを発表。20,000時間分のロボット操作データで学習して、9種類の異なるロボットで動く。「1つの脳で、どんな身体でも動かせる」方向に向かってる。

💡 なぜ「今」なのか

1. Foundation Modelの汎用化 — LLMの「大規模学習→微調整」パラダイムがロボットに移植された。1モデルで複数タスク対応、開発コスト激減。

2. Sim-to-Real — シミュレーション上で訓練して現実に一発適応。実機なしでロボットを育てられるようになった。

3. World Models — ロボットが「次に何が起きるか」を内部予測しながら動く。未知の環境への適応力が飛躍的に上がった。

🔥 So What? — で、何が重要なの

労働市場 — ホワイトカラーの次に、物流・製造・家事がAIの射程に入る。Figure 03の量産目標は年12,000台。10年後には桁が変わってる。

投資テーマ — 「AI銘柄」の意味が変わる。ソフトだけじゃなく、アクチュエータ、センサー、バッテリーへ裾野拡大。NVIDIAのロボティクス戦略はGPU制覇の再現。

日本 — 超高齢社会×人手不足の日本にとって、汎用ヒューマノイドは「移民政策に代わる解」になりうる。ファナック、安川電機がVLA時代にどう動くかが鍵。
